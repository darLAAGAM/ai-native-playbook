# Prompts Library

25 copy-paste-ready prompts across 10 domains. These prompts support the skills library and can be used directly in Claude, Codex, ChatGPT, or any internal agent system.

License: free to use, attribution requested.

## How to use

1. Pick the prompt matching the workflow you need.
2. Replace `{{placeholders}}` with your own data, policies, exports, or context.
3. Keep source facts separate from model interpretation.
4. Store strong outputs and rejected outputs in your brain so future prompts can improve.

## Prompt Hygiene

Do not paste secrets, private customer data, credentials, or regulated personal data into hosted LLMs unless your company policy and vendor contracts allow it. Prefer redacted inputs when building reusable examples.
