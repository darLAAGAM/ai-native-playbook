# Brain Starter

Copy this folder to `brain/` and fill the files in this order:

1. `knowledge/company-profile.md`
2. `knowledge/policies/returns.md`
3. `knowledge/policies/shipping.md`
4. `knowledge/operations/systems-map.md`
5. `knowledge/marketing/brand-voice.md`

Rules:

- Search the brain before answering company-specific questions.
- Cite the file used for factual answers.
- Capture temporary notes in `memory/YYYY-MM-DD.md`.
- Promote durable learnings into `knowledge/`, `skills/`, or `patterns/`.
- Do not store credentials, raw customer data, or sensitive employee records here.
