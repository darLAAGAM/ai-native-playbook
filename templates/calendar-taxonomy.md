# Calendar Taxonomy

## Sub-calendars

1. Launches and product moments
2. Marketing and content
3. Retail/channel activations
4. Supplier, production, or replenishment
5. Finance and reporting
6. People and internal operations
7. Exceptions and deadlines

## Event fields

| Field | Required | Example |
|---|---:|---|
| Type | Yes | Launch, finance, supplier, review |
| Owner | Yes | Function or person |
| Source | Yes | Sheet/doc/system |
| Confidence | Yes | Confirmed, tentative, draft |
| Visibility | Yes | Team, leadership, private |
| Action required | Yes | Yes/no |
| Source link | Yes | URL/path |

## Naming

```text
[TYPE] Short description / owner
```

Examples:

- `[LAUNCH] Spring bundle / Growth`
- `[FINANCE] VAT filing deadline`
- `[OPS] Supplier cutoff: packaging`
- `[REVIEW] Claims approval: product family`
