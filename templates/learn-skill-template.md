# Learn Skill Template

## Trigger

Use when a human or agent discovers a durable learning: bug, decision, API gotcha, policy change, workflow, incident, or reusable pattern.

## Procedure

1. Capture the learning immediately.
2. Classify it by domain and category.
3. Check whether an existing doc or skill already covers it.
4. Write to memory if temporary.
5. Promote to knowledge, skill, pattern, or lesson if durable.
