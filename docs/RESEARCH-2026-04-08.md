### 4. Missing Production Learnings (HIGH PRIORITY)

**Not yet in Ch.11b (should be added as lessons 22-28):**

1. **Lesson 22: The Anti-Prompt-Injection Hardening Rollout (3 Apr)**  
   All 7 SOULs got anti-injection sections. CS Agent got extra for CS ticket processing. Template at `knowledge/platform/agents/anti-injection-template.md`.

2. **Lesson 23: The ACK Rule — Never in Silence (6 Apr)**  
   Agents were starting work silently without confirming receipt. Every SOUL now requires immediate ACK with per-agent personality ("Enterado, lo miro" / "Voy a mirarlo" / etc.). Critical for human trust.

3. **Lesson 24: Free-Tier LLMs Don't Scale — The MiniMax Pivot**  
   Qwen 3.6 Plus free and Kimi K2.5 free were saturated with rate limits. Switched primary to MiniMax M2.5 ($0.12/M input) with paid Qwen as fallback. Stable + cheap + no rate limits.

4. **Lesson 25: OpenClaw ≠ systemd Type=simple**  
   Strategy Agent's systemd service (7 Apr) caused restart loop because OpenClaw's parent process forks a child and exits. systemd saw exit 1 → killed child → restart → loop. **Rule:** On Linux VPS use `cron @reboot`, NOT systemd for OpenClaw gateways.

5. **Lesson 26: When the API Exists But Is Broken (Richpanel)**  
   API key expired/format changed → HTTP 403 on all endpoints despite correct auth. Not every "it was working yesterday" is a code bug — sometimes the vendor silently changes auth.

6. **Lesson 27: TC Analytics + MCP Event Loop Collision**  
   TC Analytics scraper uses Playwright (asyncio). Ran inside MCP server (also asyncio) → event loop conflict. Fix: `start_new_session=True` to isolate the subprocess.

7. **Lesson 28: Heartbeat Config Schema Drift**  
   OpenClaw v2026.4+ rejects legacy keys (`intervalMinutes`, `interval`) in heartbeat config. `openclaw doctor` fails loudly. Fix: reduce to only `every` + `activeHours`.

**Action:** Add 7 new lessons to Ch.11b. Re-number if needed.
