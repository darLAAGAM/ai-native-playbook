# Monitor Ad Pacing

## Purpose

Project month-end spend and recommend daily budget changes before over/underspend becomes expensive.

## Use When

Use when you need to project month-end spend and recommend daily budget changes before over/underspend becomes expensive.

## Inputs

- Monthly budget by channel/campaign
- Month-to-date spend
- Remaining days
- Performance target

## Procedure

1. Calculate current daily run rate and projected month-end spend.
2. Compare projection to target by channel and campaign.
3. Flag overpacing, underpacing, and campaigns constrained by bid, audience, budget, or learning status.
4. Compute daily spend needed for each campaign to land on target.
5. Recommend reallocation only when performance quality supports it.

## Output

- Pacing table
- Projected spend
- Required daily spend
- Constraints
- Recommended changes

## Guardrails

- Do not force spend into poor-performing campaigns just to hit budget.
- Separate budget delivery risk from efficiency risk.

## How to adapt

Customize by calendar, promo periods, weekday/weekend patterns, and finance controls.
