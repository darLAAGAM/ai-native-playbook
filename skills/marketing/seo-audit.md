# SEO Audit

## Purpose

Audit a site for crawlability, indexation, on-page quality, content depth, and authority gaps.

## Use When

Use this skill when an operator needs a repeatable procedure for audit a site for crawlability, indexation, on-page quality, content depth, and authority gaps.

## Inputs

- Website or URL list
- Priority keywords/pages
- Search Console data if available
- Competitors

## Procedure

1. Start with crawlability and indexation: robots, sitemap, canonicals, noindex, redirects, and coverage.
2. Review technical foundations: Core Web Vitals, mobile usability, HTTPS, URL structure, broken links.
3. Audit on-page elements: titles, descriptions, H1s, headings, internal links, image alt text, keyword mapping.
4. Assess content quality and search intent match against competitors.
5. Identify authority gaps and structured data opportunities.
6. Prioritize fixes by impact and effort.

## Output

- Executive summary
- Technical findings
- On-page findings
- Content findings
- Prioritized action plan

## Guardrails

- Do not treat SEO tools as truth without spot checks.
- Separate blocker fixes from nice-to-have optimization.

## How to adapt

Adapt priority scoring to your site size, tech stack, and commercial pages.
