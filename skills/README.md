# Skills Library

75 skills across 13 domains. Each file contains a defined procedure, inputs, outputs, guardrails, and adaptation notes for consumer SME operators.

License: free to use, attribution requested.

## How to use

1. Mount this repo as part of your company brain or agent workspace.
2. Let any Claude/Codex/AI client with repository access read these skills.
3. Invoke a skill by name from any chat session.
4. Adapt each skill by editing the `How to adapt` section and replacing thresholds, owners, tools, and policy details.
