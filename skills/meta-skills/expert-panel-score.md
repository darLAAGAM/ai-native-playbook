# Expert Panel Score

## Purpose

Score an artifact with multiple expert lenses and iterate until it passes the quality bar.

## Use When

Use this skill when an operator needs a repeatable procedure for score an artifact with multiple expert lenses and iterate until it passes the quality bar.

## Inputs

- Artifact
- Audience
- Goal
- Brand or quality rules
- Pass threshold

## Procedure

1. Identify the artifact type and assemble 5-10 relevant expert lenses.
2. Define a rubric with weighted criteria and pass threshold.
3. Score the artifact independently through each lens.
4. List the top weaknesses and revise the artifact.
5. Repeat until it passes or the maximum rounds are reached.
6. Save the final version and the scoring history.

## Output

- Final artifact
- Score table
- Weaknesses fixed
- Remaining risks

## Guardrails

- Do not inflate scores to pass.
- Show enough critique for the user to trust the result.

## How to adapt

Customize rubrics, expert lenses, and score thresholds for each content or decision type.
