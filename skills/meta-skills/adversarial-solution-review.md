# Adversarial Solution Review

## Purpose

Generate options, attack them, repair them, and consolidate a stronger recommendation.

## Use When

Use when you need to generate options, attack them, repair them, and consolidate a stronger recommendation.

## Inputs

- Problem
- Constraints
- Success criteria
- Known options

## Procedure

1. Generate 3-7 distinct solution approaches.
2. For each approach, identify failure modes, hidden assumptions, costs, edge cases, and unintended consequences.
3. Propose fixes for the strongest critiques.
4. Validate whether the fixes solve the root issue or create new problems.
5. Consolidate overlapping options and rank the viable ones.
6. Recommend the top path with metrics and warning signs.

## Output

- Detailed critique
- Ranked options
- Final recommendation
- Watch metrics

## Guardrails

- Use for complex problems, not obvious tasks.
- Be explicit when all options are weak.

## How to adapt

Adapt evaluation criteria to technical, operational, financial, or strategic decisions.
