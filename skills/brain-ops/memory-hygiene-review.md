# Memory Hygiene Review

## Purpose

Keep daily memories useful by compressing noise and preserving decisions.

## Use When

Use this skill when an operator needs a repeatable procedure for keep daily memories useful by compressing noise and preserving decisions.

## Inputs

- Daily memory files
- Decision records
- Action logs

## Procedure

1. Scan recent memory for decisions, facts, unresolved tasks, incidents, and learnings.
2. Move durable knowledge into stable domain docs.
3. Archive or summarize noisy session logs.
4. Link decisions to the source evidence and owner.
5. Flag stale, duplicated, or contradictory entries.

## Output

- Stable knowledge updates
- Archived noise
- Contradiction list
- Open follow-ups

## Guardrails

- Do not delete without preserving useful context.
- Do not promote unverified claims to stable docs.

## How to adapt

Set a cadence, retention policy, and what qualifies as durable knowledge.
