# Variant Or Size Gap Analysis

## Purpose

Find broken assortments where demand exists but key variants/sizes are missing.

## Use When

Use this skill when an operator needs a repeatable procedure for find broken assortments where demand exists but key variants/sizes are missing.

## Inputs

- Variant-level inventory
- Variant-level sales
- Returns by variant if available
- Product page views if available

## Procedure

1. Calculate availability and sell-through by variant/size/color/location.
2. Find products with strong demand but missing core variants.
3. Separate true preference from availability bias.
4. Recommend replenishment, transfers, substitution messaging, or product page changes.
5. Flag variants that over-index in returns or complaints.

## Output

- Gap table
- Demand signal
- Recommended action
- Risk

## Guardrails

- Do not conclude customers dislike a variant that was rarely available.
- Check returns and complaints before reordering aggressively.

## How to adapt

Map variant names to your category logic: size, color, flavor, scent, material, pack size, or configuration.
